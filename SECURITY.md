# Security Policy

## Reporting a vulnerability

If you believe you have found a security vulnerability, please report it privately.

- Email: the maintainer email listed in `pyproject.toml`

Please include:

- A description of the issue
- Steps to reproduce
- Impact assessment (what an attacker could do)
- Any suggested mitigations

## Supported versions

This project is currently in **alpha**. Security fixes will be provided on a best-effort basis for the latest released version.
